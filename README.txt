
# **Kundai Zhou – Portfolio Website**  

## Overview  
This is my **personal portfolio website**, showcasing my work as a **WordPress Web Designer and Developer**. It highlights my skills, projects, and services, providing a central hub for potential clients and collaborators to learn more about what I offer.  

## Features  
- Fully **responsive** and optimized for all devices  
- Modern **UI/UX design** with a clean layout  
- **Project showcase** with case studies  
- Contact links for easy inquiries    
- Social media and GitHub links  

## Tech Stack  
- **Frontend:** HTML, CSS, BasicJavaScript  
- **Backend:** WordPress (PHP, MySQL)  
- **Plugins & Tools:** Elementor, Advanced Custom Fields, Rank Math SEO, WooCommerce, SureCart  
- **Design Tools:** Figma, Canva
